# Contributing to HostCore Mobile Cloud

First off, thanks for taking the time to contribute! 🎉

The following is a set of guidelines for contributing to HostCore Mobile Cloud.

## How Can I Contribute?

### Reporting Bugs
- Use the Bug Report template.
- Provide as much detail as possible (Android version, Termux version, etc.).

### Suggesting Enhancements
- Use the Feature Request template.
- Explain why this enhancement would be useful.

### Pull Requests
- Fill out the Pull Request template.
- Ensure your code passes CI tests.
- Follow the existing code style.

## Style Guide
- Use 4 spaces for indentation.
- Keep functions modular and well-documented.
- Write clear, descriptive commit messages.

Thanks!
